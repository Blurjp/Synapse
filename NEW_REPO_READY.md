## 🚀 可以开始了！

### 克隆仓库

```bash
git clone https://github.com/Blurjp/Synapse.git
cd Synapse
```

### 设置前端

```bash
cd frontend
npm install
npm run dev
```

### 设置后端

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8001
```

### 安装 Extension

1. 打开 `chrome://extensions/`
2. 启用 "Developer mode"
3. 点击 "Load unpacked"
4. 选择 `extension` 文件夹
