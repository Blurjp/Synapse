### 📖 How to Use

**1. Save Webpages**
- Click the Synapse icon in your toolbar
- Press "Save to Synapse"
- Or use keyboard shortcut: `Ctrl/Cmd + Shift + S`

**2. Save Highlights**
- Select any text on a webpage
- Right-click → "Save Highlight to Synapse"
- Add notes to your highlights

**3. Take Quick Notes**
- Click the Synapse icon
- Type your note in the text area
- Press "Save Note"

### ⚙️ Setup Requirements

**This Extension requires the Synapse backend:**

1. **Install Synapse Backend** (One-time setup)
   ```bash
   git clone https://github.com/Blurjp/Synapse
   cd Synapse/backend
   pip install -r requirements.txt
   uvicorn app.main:app --port 8001
   ```

2. **Load the Extension**
   - The Extension will automatically connect to localhost:8001

**Full setup guide:** https://github.com/Blurjp/Synapse
